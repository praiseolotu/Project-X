Variables

You use variables as symbolic names in your application. The name of variables, also called identifiers conform to certain rules.

In Python, variables names starts with lowercase, and if your variable name has more than one word, then you use camel case notation or you separate the words with an underscore (_).

# I declared an integer varaible a int a # I declared a double varaible b double b
Declaring variables in Python
You can declare variables in two ways:

You can just declare the variables straight up: b = 42 Python knows that you are assigning the integer 42 to b.

You can specify the variable datatype beforehand: int b = 42 We are telling Python that variable b can only store an integer.

Variable Scope

When you declare a variable outside any function, then that variable is a global variable. But when you declare a variable inside a function, then that variable is a local variable.

                d = 3.142 # global variable
                    
                def func():
                    c = 14 # local variable
                    print("Local Variable")

if...else Statement

Use the if statement to execute a logical statement if condition is true. Use the exception else statement to execute a logical statement if condition is false.

                    if condition:
                        statement
                    else:
                        statement

You may also compound the statement using elif if you have multiple conditions to test. In the case of multiple conditions only the first logical condition which evaluates to true will be executed

                if condition:
                    statement
                elif condition:
                    statement
                elif condition:
                    statement
                else:
                    statement

You can only have only 1 if and else with multiple elif conditions.

while Statement

A while statement executes its statements as long as a specified condition evaluates to true. A while statement looks as follows:

                while condition:
                    statement

If the condition becomes false, statement within the loop stops executing and control passes to the statement following the loop. The condition test occurs before statement in the loop is executed. If the condition returns true, statement is executed and the condition is tested again. If the condition returns false, execution stops and control is passed to the statement following while.

To execute multiple statements, you use identation.

                while condition:
                    statement1
                    statement2
                    statement3
                    statement4
                    statement5

  • def keyword
  • The name of the function.(The name must start with lower case and if it contains multiple words, then use camel case notation)
  • A list of arguments to the function, enclosed in parentheses and separated by commas
  • A colon(:)
